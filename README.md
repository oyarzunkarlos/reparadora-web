# 🔧 Reparadora de Calzados y Ortopedia - Sitio Web

Sitio web desarrollado como proyecto de aprendizaje para la transición a FullStack Developer.

## 📚 Estado del Proyecto

**Fase 1:** Fundamentos (HTML5 + CSS3 + JavaScript + Git)

## 🛠️ Tecnologías

- **HTML5** semántico
- **Tailwind CSS** (vía CDN para desarrollo)
- **JavaScript ES6+** (vanilla, sin frameworks)
- **Git** para control de versiones

## 📁 Estructura del Proyecto

```
reparadora-web/
├── index.html          # Página principal
├── main.js             # Lógica JavaScript
├── README.md           # Este archivo
└── assets/             # Imágenes y recursos (pendiente)
```

## 🚀 Cómo usar este proyecto

1. Clonar el repositorio
2. Abrir `index.html` en el navegador o usar Live Server
3. ¡Listo!

## 📝 Ejercicios de Aprendizaje

Revisa el archivo `main.js` - incluye comentarios con:
- Código base implementado
- **Tareas pendientes** para que practiques
